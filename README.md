# <情報共有コミュ>

## サイト概要
### サイトテーマ
* 仕事をする上での情報共有をジャンル別で確認できるようにした
コミュニティサイト


### テーマを選んだ理由
* これまでの職業でどれも難しかったのが情報共有でした。
共有する情報の整理ができず、共有漏れや確認漏れ、誰が対応したか等を
一括で確認できるようにしたと考えたので、このテーマにしました。

### ターゲットユーザ
* 情報共有で悩んでる方、チームで働く全ての人へ

### 主な利用シーン
* 情報共有をしたい時、共有内容を確認したい時
